O que significa web semântica?

A Web Semântica tenta, justamente, acabar com essa confusão.
Ela se trata de um projeto de aplicação de conceitos inteligentes na internet, a partir de padrões de formatação de dados.
A ideia é que a máquina consiga ler um bloco inteiro de informação, não apenas uma palavra solta.

https://blog.xpeducacao.com.br/web-semantica/

Qual é a proposta da Web Semântica?

A Web Semântica busca facilitar o processo de comunicação entre os diversos participantes do ecossistema,
de forma a criar um modelo mental comum, minimizando a possibilidade de ambiguidades, e facilitando, assim, o trabalho.
